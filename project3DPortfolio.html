Why 3D is Necessary for this Project

This project is significantly **better as a 3D experience** than as a static image or video for the following reasons:

  • **Spatial Presence:** The "Oracle" is designed as a spatial environment. The user must navigate around the elements, allowing the geometry to shift and reveal new visual patterns based on perspective, which a 2D image cannot capture.
  • **Immersion and Scale:** Using A-Frame creates a sense of scale and depth. The user is *in* the data stream or digital space, enhancing the feeling of being present with the "Oracle," which is key to the conceptual art installation goal.
  • **Interactive Discovery:** The user's movement and viewing angle directly influence the experience. This active discovery is central to the project and requires the full rotational freedom provided by the 3D environment.
