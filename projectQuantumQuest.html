Quantum Quest: The Adventure Game

Quantum Quest was created as an Adventure Game project. It was a technical challenge to implement the final state of the game which is the Boss fight. It was my first experience designing a game in this format.

Game Narrative and Mechanics

The game explores several game design and adventure game mechanics. The player must use key strokes and mouse clicks in order to to progress through the world and reach the final win state.

This project was crucial for developing my skills in p5.js developing and state management.
